# treinaweb-express

Projeto base com Node.js, Express e TypeScript.

## Documentacao

- [Setup do projeto](./docs/setup.md)

## Scripts

- `npm run dev`: executa a aplicacao com `tsx`
- `npm run watch`: executa com `ts-node-dev` e reinicia ao salvar
- `npm run build`: compila o TypeScript para `dist`
- `npm start`: compila e sobe a versao de producao
